# 智见 Lite - 仓储质检 AI 系统

智见 Lite 是面向制造业仓储现场的 AI 质检系统。系统围绕入厂收货、工位检测、盘点复查、出库复核、退料复检、问题件移交和索赔取证等工作，把 PDA 手持端、Station 检测工位和 Admin 管理后台放在同一套业务流程里。

当前仓库交付的是 MVP Web 版本，重点验证三端交互、现场任务流、异常处理、AI 建议呈现和后台配置方式。它不是一个只停留在前端页面的产品定义；正式版本会接入现场设备、视觉识别、OCR、条码/RFID、仓储系统、权限审计和后端服务。

## 当前版本已实现

### 首页

- 展示智见 Lite 的产品介绍、核心能力、业务场景和关键指标。
- 提供 PDA、Station、Admin 三个入口，便于按现场角色进入对应界面。
- 通过真实仓储质检语境组织页面内容，而不是只展示孤立组件。

### PDA 手持端

入口：`/#/pda`

PDA 端面向现场操作人员，覆盖移动作业中的扫码、核验、上报和交接动作。

- 收货扫码、收货明细核验、收货结果摘要、到货预约登记。
- 上架任务、上架扫码、拣货任务、拣货扫码。
- 巡检任务、巡检执行、异常上报、问题件移交。
- 盘点复查、盘点结果处理、标签补打。
- 生产退料扫码、退料明细核验。
- 防伪溯源、个人信息、设备状态和当日工作摘要。

### Station 检测工位

入口：`/#/station`

Station 端面向固定检测工位，呈现相机识别、人工确认、告警和处理回执等工作。

- 待机页：展示今日任务、设备状态、检测摘要和 Agent 提醒。
- 收货检测：支持条码、OCR、标签、外观和数量等检测状态。
- 出库复核：按出库单、包裹、标签和异常提醒组织复核流程。
- 盘点复查：支持库位、物料、数量和标签的复查展示。
- 退料复检：呈现退料件识别、复检结论和处理建议。
- 异常分流：展示问题件编号、责任判断和分流处理。
- 检测画面内包含识别框、状态切换、AI 分析面板和协作回执。

### Admin 管理后台

入口：`/#/admin`

Admin 端面向仓储主管、质量管理和系统管理员，负责查看事件、配置规则和跟进处理结果。

- 首页工作台：今日数据、待处理事项、Agent 提醒和快捷入口。
- Agent 协作台：查看事件列表、事件详情、建议和处理结果。
- 工单中心：管理质检、复核、上报和处理类工单。
- 数据看板：查看质量指标、异常趋势和 AI/人工处理对比。
- 规则配置：维护检测规则、拦截规则和建议规则。
- 供应商分析：查看供应商异常记录和质量表现。
- 异常案例库：保存图片、编号、原因和处理记录。
- 质量标准库：查询品类、物料和场景对应的判定标准。
- 问题件/隔离区：管理待处理问题件、滞留时长和责任信息。
- 索赔协同：组织照片、监控截图、责任方和沟通记录。
- 盘点策略：配置动碰复查、静态复查和重点库位复查。
- 视频违规分析：查看违规时间线、截图证据和处理建议。
- 采购计划对接：查看收货偏差、拦截记录和采购计划影响。
- 入库、出库、退料相关配置页：管理关键流程中的规则和展示项。

## MVP 说明

当前版本以 Web 方式呈现产品的主要工作界面和关键流程，方便快速验证现场使用方式、页面结构和异常处理路径。

- 当前版本使用 React Context 和 localStorage 管理页面状态。
- 当前版本内置演示样例，用于呈现业务流程和状态变化。
- 正式版本会由后端 API、设备网关、AI 服务和业务系统提供数据。
- 摄像头、PDA 扫码、条码/RFID、打印、补光、称重等设备会在正式落地时接入。
- 图像缺陷检测、OCR、包裹数量核对、标签识别和视频违规分析会接入实际模型服务。
- WMS、ERP、MES、采购计划和供应商质量记录会通过接口同步。
- 登录、角色权限、操作日志、告警记录和证据文件存储会作为正式系统能力建设。

## 系统目标

- 减少收货、盘点、出库和退料中的人工漏检与重复录入。
- 把图片、标签、条码、单据、责任判断和处理记录保存在同一事件中。
- 区分强制拦截、处理建议和人工确认，让现场人员知道下一步该做什么。
- 让管理人员能追踪异常来源、处理进度、供应商质量和索赔依据。
- 为后续接入现场设备、后端服务和企业业务系统保留清晰接口。
